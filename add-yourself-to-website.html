Add Yourself to the Website

a quick introduction to working with github

  1. Login or create a GitHub account
  2. Visit https://github.com/CodeForCharlotte/codeforcharlotte.org
  3. Click Fork
  4. Click the data folder
  5. Click the team.yml file
  6. Click the pencil icon on the far right in the header above the line numbers
  7. Click into the edit area at the last line after the last person’s entry
  8. Type tab twice, a hyphen and return
  9. Type tab once, name followed by a colon, a space, then your name in quotation marks and return
  10. If you have a Gravatar account, Type email followed by a colon, a space, then the email address and return
  11. If you don’t want to display a picture of yourself, Type email followed by a colon, a space and return
  12. Type bio followed by a colon, a space then about something of yourself in quotation marks and return
  13. Click Commit changes after entering in a summary of the changes and more detail if desired
  14. Click Pull Requests on the far right
  15. Click the create a pull request link
  16. Click Create pull request
  17. Click Create pull request
  18. Someone will check the format is correct and merge your pull request and you’ll be added to the website!
