How to Contribute

About our website

This website is constructed using the static-site generator technology Middleman. Middleman helps make site maintenance easier by allowing for many programming techniques to be used to produce the HTML, rather than relying on a particular editor that might only be available for Windows or Mac, for example. Middleman is open-soure, and written in Ruby, an open-source language. We think it's just right!

We also host our project on GitHub and use git for source control.

The current look-and-feel is powered by Twitter Bootstrap, and we use Font Awesome in a couple of places.

This page describes how install middleman, and how to make contributions to this site.

Installing Middleman

Windows

  • Download The Windows Ruby 1.9 build here. This download includes git.
  • That should create a "Ruby on Rails Console" program in your start menu. Run that. At a command line, type gem install middleman bundler.
  • You'll need a text editor. We recommend the Sublime Text editor, although any plain-text editor would be fine.

OS X

  • Download XCode from the App Store (free). Install the command-line utilities from the Preferences -> Downloads menu. You may also be able to issue the command xcode-select --install in your Terminal program.
  • Open up Terminal from Applications/Utilities, and install Homebrew
  • brew install git
  • Follow the Linux instructions to install ruby.

Linux

  • Follow the "Basic GitHub Checkout" installation instructions for rbenv.
  • Install ruby-build as a plugin
  • rbenv install 2.1.2 -- as of the current date, 2.1.2 is the most recent ruby available from ruby-build
  • gem install middleman

Building the website

To acquire the website, you'll do something like this: 

                  
                    mkdir Sites
                    cd Sites
                    git clone git@github.com:CodeForCharlotte/codeforcharlotte.org.git
                    cd codeforcharlotte.org
                    bundle
                    bundle exec middleman server

Depending on your Github setup, that last line might vary. View the repo and reference those clone URL's to make sure that you're using the right one for you.

After the above steps, you'll be able to load this page and see a version of this page! As long as the middleman server is running, any changes that you make to source pages will be available after a reload.
